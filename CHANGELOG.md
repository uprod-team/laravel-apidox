### Added
- Initial release.
- `RouteScanner` that discovers routes matching configurable prefixes.
- PHPDoc parser (summary + description).
- FormRequest rules → body parameters table with type inference.
- Auth & rate-limit detection from middleware.
- Embedded zero-dependency CSS (~10KB, Trust & Authority style).
- Webhooks documentation section with HMAC signature verification samples.
- Publishable config and views.
- Tests on Laravel 11/12/13 and PHP 8.2/8.3/8.4.
